# Purpose of the Prompt

This AI prompt helps QA Automation Engineers generate comprehensive test suites from user stories. It creates detailed test cases across multiple testing categories including functional, negative, boundary, security, and more, ensuring consistent coverage and providing actionable test cases with proper structure and priority levels.

# Problem It Solves

Manually analyzing requirements can miss scenarios and result in inconsistent test coverage. The prompt improves consistency and coverage by systematically generating test cases across all relevant testing dimensions, ensuring that critical scenarios are not overlooked and that test suites follow a standardized format.

# Usage Frequency

I use this prompt several times per week when analyzing user stories and preparing QA scenarios for new features and regression testing.

# Repository Contents

- **prompts folder:** Contains the AI prompt used to generate test suites
- **evaluations folder:** Contains evaluation documents demonstrating the prompt working with different user story scenarios
- **examples folder:** Contains example user stories used to evaluate the prompt
