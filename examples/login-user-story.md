# Login User Story

## Title
User Login

## Description
As a registered user,

I want to log into the application using my email address and password,

So that I can securely access my account and personalized features.

## Acceptance Criteria

1. The user can log in using a valid email address and password.
2. An error message is displayed when invalid credentials are entered.
3. Both Email and Password fields are required.
4. The password field masks the entered characters.
5. The user is redirected to the Dashboard after a successful login.
6. The application remembers the authenticated session until the user logs out.
7. The login page is accessible from supported desktop and mobile browsers.
